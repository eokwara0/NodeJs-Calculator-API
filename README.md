# NodeJs-Calculator-API
NodeJS calculator API that allows
  Math Operatoions
   * ADD 
   * SUBTRACT
   * MULTIPLY
   * DIVIDE
   

## OPERATIONS
  
  ### ADD
    
    * Query based route
    * Query ID => ( values ) followed by number's seperated by ( , )
    * Route => 'http://localhost:8080/calculator/add?values=<number>,<number> .... '
    
    
 ### SUBTRACT
